# Apex Regex Tester

A dead-simple, fast, and ad-free web tool for testing regular expressions. Includes a library of common patterns with clear explanations.

**Features:**
- Real-time regex testing
- Input string and regex pattern fields
- Pre-built common regex patterns (email, URL, etc.) with explanations
- Clean, minimalist UI
- One-time purchase for lifetime access (Stripe integration coming soon)

Built for developers who need a quick, distraction-free way to work with regex.
